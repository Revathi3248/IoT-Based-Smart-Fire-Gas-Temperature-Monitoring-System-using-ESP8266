# IoT-Based-Smart-Fire-Gas-Temperature-Monitoring-System-using-ESP8266
This project is an IoT-based safety monitoring system using ESP8266, DHT11, flame, and gas sensors. It monitors temperature, humidity, fire, and gas leakage in real time and publishes data to Adafruit IO using MQTT. Automated actions like fan, pump, buzzer, and LED alerts are triggered during emergencies.
